

    Doctor Reservation API

    Backend API to handle reservation between doctors and patients.

    This API introduce links to:
        - add new doctor,
        - update if existed,
        - view specific doctor
        - view all added doctors
        - remove existed doctor.

    Also, you can find links like this with patient and appointment entities.
    an addition to link to count doctor appointments and another one for patient.

    You can use Swagger API from link below to show all of this:
        http://localhost:8080/swagger-ui/index.html
